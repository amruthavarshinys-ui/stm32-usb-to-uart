# STM32 USB to UART Converter (Work in Progress)

## Overview
This project is a USB to UART converter designed using an STM32 microcontroller.  
The goal is to enable communication between a computer (USB) and UART-based devices.

## Features
- STM32-based design
- USB communication interface
- UART output pins
- 3.3V power regulation
- Custom PCB designed using KiCad

## Tools Used
- KiCad (for schematic and PCB design)
- STM32 ecosystem

## Project Files
- Schematic (.kicad_sch)
- PCB Layout (.kicad_pcb)
- Project file (.kicad_pro)

## Current Status
